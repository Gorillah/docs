# Introduction

Welcome to the Blurr API documentation! Blurr is a cutting-edge platform offering AI-powered calling solutions for both inbound and outbound communications. Our API allows you to integrate our advanced calling features into your applications seamlessly, enabling automated and intelligent interactions with your customers.

## What You Can Do with Blurr API

With the Blurr API, you can:

- **Initiate Outbound Calls**: Automate outbound calls to your customers for various use cases such as marketing campaigns, reminders, and notifications.
- **Handle Inbound Calls**: Use AI to manage and respond to incoming calls, providing your customers with quick and accurate information.
- **Customizable Interactions**: Tailor the conversation flow to suit your business needs, ensuring a personalized customer experience.
- **Track and Monitor Calls**: Access detailed call logs and analytics to monitor performance and improve your calling strategies.

## Getting Started

To start using the Blurr API, follow these simple steps:

1. **Sign Up**: Create an account on our [Blurr website](https://www.blurr.app/auth/signup).
2. **Get Your API Key**: Once logged in, navigate to the Settings -> API section to generate your API key.
3. **Read the Documentation**: Familiarize yourself with our API endpoints, request parameters, and response formats.

There are two ways to build API documentation: [OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup) and [MDX components](https://mintlify.com/docs/api-playground/mdx/configuration). For the starter kit, we are using the following OpenAPI specification.

## Authentication

The Blurr API uses Bearer Token authentication. Include your API key in the `Authorization` header of your requests as follows:

```http
Authorization: Bearer YOUR_API_KEY
```
